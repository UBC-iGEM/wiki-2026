# Purpose

The purpose of this entry is to perform HiFi assembly for Cpxp and recA.

# Materials & Methods

HiFi assembly protocol taken from NEB.
For a 2-3 fragment HiFi assembly at a 1:2 vector-insert ratio
using suggested min 0.03 pmol for total fragments
(note MM is always 1/2 total rxn vol)
Scale down to 6 uL by multiplying by 0.3 (6/20) (vol in uL)

|                   |        |        |
| ----------------- | ------ | ------ |
| Content           | Cpxp   | recA   |
| Insert            | 0.3486 | 0.3186 |
| Backbone          | 0.3843 | 0.3843 |
| HiFi MM           | 3      | 3      |
| Nuclease free H2O | 2.2671 | 2.2971 |
| Total             | 6      | 6      |
|                   |        |        |
